財富和快樂的對應公式
    每月基本開銷: EX:台北單身狗(NT28000[USD980])
    快樂基本門檻=收入大於『每月基本開銷』
    何謂便宜商品 = 每月收入/100[一般人會花上薪資的10%用在犒賞自己(也就是非必需品)]
    何謂貴的商品(奢侈品) =每月收入*1/3
    每月賺多少錢才能快樂=『每月基本開銷』*5
    薪水高低VS快樂指數轉換公式=目前每月薪水/基本開銷=0~5(每0.5為一個間隔，可分成10個等分)
